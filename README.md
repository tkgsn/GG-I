# GG-I
This implementation is written by Python 3.7 with libspatialindex.


## How to run
```
git clone https://github.com/tkgsn/GG-I.git
cd GG-I

python -m venv venv
. venv/bin/activate
pip install -r requirements.txt

jupyter lab
```

## Experiments
run_time.ipynb
- You can run optimization.

exp_in_real-world_map.ipynb
